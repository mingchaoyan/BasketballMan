投篮游戏
============

玩法说明
---------

3D投射游戏，使用上下左右方向键或者wasd键控制运动员方向，使用数字键1投篮。游戏目的在规定时间60秒内，投中越多获得越高分。

技术细节
--------
基于Unity3d 4.3.4f1，使用最原始的GUI制作简单的界面按钮。用简单的物理碰撞处理篮球，篮板和地面之间的撞击，投篮动作使用animation动画分割播放实现。是否投中使用在篮框中放置一个虚拟的碰撞体来“trigger”。
